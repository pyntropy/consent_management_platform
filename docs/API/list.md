GET Получить список всех согласий на подпись (может быть собрано несколько бандлов)

POST /processing/consents Принять согласия
Принимаются конкретные согласия, а СМР проверяет активные бандлы на подписании.

GET /consents Получить список принятых согласий

POST Отозвать согласие
POST Отозвать все согласия

Инициализация процесса:
POST /doc/signing — создать сборку документов согласий на подпись.
GET /doc/consents?bundleId=<id> — получить документы согласий на подпись по id сборки (события).
GET /doc/consents


POST /doc/bundle/<id>/sign-command — отправляет на подпись документ согласия на подпись.

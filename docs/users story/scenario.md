# Запрос списка атрибутов дял формирования согласий

(1) Канал отправляет в СРМ атрибуты бизнес-кейса.
Частный случай: id продукта.
GET /bundle/attrs&product_id=1

(2) СМР формирует список атрибутов ПДн которые необходимо передать в СМР для формирования согласий.

(2.1) СМР выбирает в таблице атрибутов бандлов соответствующий id бандла.
SELECT bundle_id FROM bundle_attrs ...

(2.2) СМР выбирает в таблицах согласий все связанные с бандлом id согласий.
SELECT consent_id FROM bundle_consents WHERE bundle_id=(2.1)

(2.3) СМР выбирает в таблице согласий последние версии согласий
SELECT * FROM consents WHERE id IN (2.2)

(2.4) СМР выбирает из таблицы consent_attrs все атрибуты необходимые для формирования согласий.


те согласия которые указаны в цепочках как последняя версия.

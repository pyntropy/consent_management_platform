# Список user story для роли "Оператор системы СМР"

`OPER-01` Я как оператор системы СМР хочу добавить новое согласие, 
чтобы добавлять его в сборки согласий для различных бизнес-сценариев
и дать возможность каналам принимать согласия клиентов.

`OPER-02` Я как оператор системы СМР хочу добавить новое согласие, 
чтобы добавлять его в сборки согласий для различных бизнес-сценариев
и дать возможность каналам принимать согласия клиентов.

`OPER-03` Я как оператор системы СМР хочу добавить к согласию которые требуют документа,
шаблон документа, чтобы пользователь мог ознакомиться с условиями согласия.

`OPER-04` Я как оператор системы СМР хочу добавить сборку согласий для определенного бизнес-сценария,
чтобы в момент наступления события, канал сбора согласий мог запросить требуемые документы.

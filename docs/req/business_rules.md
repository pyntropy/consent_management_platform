# Бизнес-правила

1. По законодательству РФ компании
обязаны собирать согласия на обработку персональных данных.
2. Сбор согласий регламентируется 156-ФЗ.
3. Согласие заполняется персональными данными клиента.
4. Набор персональных данных указываемых в согласии регламентирован законом 156-ФЗ.
5. Данные для взрослого человека
6. Данные для ребенка
7. Данные для получения рекламы (ограниченный набор реквизитов)

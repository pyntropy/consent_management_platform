# Добавление согласия

| Атрибут     | Значение                                              |
|:------------|:------------------------------------------------------|
| Название    | Добавление согласия в систему.                        |
| Цель        | Добавление согласия для привязки к сборкам согласий.  |
| Предусловия | Нет                                                   |
| Триггеры    | Администратор перешл на страницу добавления согласия. |
| Последствия | Согласие добавлено в систему.                         |

## Сценарий
### Основной поток
(1) Администратор перешел на страницу добавления согласия.

(2) На страницу подгружается справочная информация:
- виды наборов данных
- цели сбора согласий

(3) Администратор вводит данные согласия и нажимает кнопку "Добавить согласие".

(4) Система добавляет согласие в базу данных.

(5) Система перенаправляет администратора на страницу со списком согласий.


## Реализация

(1) UI-01: Страница GET /consents/add

(2) DB-01: data_set

(2) DB-02: data_set_attrs

(3) UI-02: POST admin/consents

(4) UI-02: Страница GET /consents/<id>

# Документ (document)

Документ согласия — и есть документ.

Документ может быть:

- фиксированный (fix, fixed), не требующий вставки данных клиента.
- переменный (var, variable), требующий вставки данных.

Документ содержит ссылки на 2 файла:

- PDF — для предпросмотра и подписания
- WORD — для заполнения.

## Операции

- C - создается и существует только в рамках согласия.
- R - чтение доступно без ограничений.
- U - документ не изменяем.
- D - удалить можно только документ, который не подписан в рамках согласия.

## Описание таблицы

| attribute     | type     | nullable | description                                         |
|---------------|----------|:--------:|-----------------------------------------------------|
| id            | UUID     |    NO    | id согласия.                                        |
| created_at    | DATETIME |    NO    |                                                     |
| updated_at    | DATETIME |    NO    |                                                     |
| deleted_at    | DATETIME |   YES    |                                                     |
| created_by    | UUID     |    NO    |                                                     |
| update_by     | UUID     |    NO    |                                                     |
| deleted_by    | UUID     |   YES    |                                                     |
| file_link_fix | TEXT     |   YES    | Ссылка на предпросмотр документа. *.pdf             |
| file_link_var | TEXT     |   YES    | Ссылка на шаблон документа с плейсхолдерами. *.docx |
| major_ver     | INTEGER  |    NO    | Мажорная версия документа.                          |       
| minor_ver     | INTEGER  |    NO    | Минорная версия документа.                          |

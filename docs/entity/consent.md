# Согласие (consent)

**Согласие** — основная сущность системы, документ, в котором указаны условия
использования компанией персональных данных клиента.

- **Дать согласие** — действие клиента.
- **Получение (сбор) согласия** — Действие системы или сотрудника компании
  по предоставлению клиенту согласия на ознакомление, и самому получению согласия в виде подписи на распечатанном
  документе
  или установки чек-бокса в приложении.
- **Принять согласие** — действие системы по фиксации (сохранению) факта согласия клиента
  с условиями использования его персональных данных условий.

Согласия версионируются.
Каждое согласие по существу представляет собой цепочку обновлений.
При получении согласия клиенту предоставляется последняя версия.
При мажорном обновлении версии ранее данного согласия клиенту прелагается принять новую версию.

Собираются в рамках сборок (bundle)

| attribute | type | description |
|-----------|------|-------------|
| id        | UUID | consent     |

# Просмотр списка согласий

| Атрибут     | Значение                                            |
|:------------|:----------------------------------------------------|
| Код         | OPER-01                                             |                                                 |
| Название    | Просмотр списка согласий.                           |
| Цель        | Просмотр списка согласий.                           |
| Предусловия | Оператор авторизован в системе.                     |
| Триггеры    | Оператор нажал на кнопку "Согласия" в главном меню. |
| Последствия | Нет.                                                |

## Сценарий

### Основной поток

(1) Браузер запрашивает страницу "Согласия".

(2) CMP выбирает список согласий, и формирует страницу.

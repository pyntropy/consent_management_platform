# Список user story для роли "Оператор системы СМР"

`OPER-01` Я, как оператор СМР, хочу добавить в СМР сборку согласий для определенного бизнес-сценария,
чтобы в момент наступления бизнес-события, канал сбора согласий мог запросить требуемые документы.

`OPER-02` Я, как оператор СМР, хочу добавить в СМР новое согласие (набор атрибутов согласия), 
чтобы добавить его в сборку (-ки) согласий.

`OPER-03` Я, как оператор СМР, хочу добавить в СМР шаблон документа согласия, 
чтобы пользователь мог ознакомиться с условиями согласия.

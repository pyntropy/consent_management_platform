# Создание нового согласие в СМР

| Атрибут     | Значение                                                                    |
|:------------|:----------------------------------------------------------------------------|
| Код         | OPER-01                                                                     |
| Название    | Создание нового согласие в СМР.                                             |
| Цель        | Добавление согласия для привязки к сборкам согласий.                        |
| Предусловия | Нет                                                                         |
| Триггеры    | На странице списка согласий администратор нажал кнопку "Добавить согласие". |
| Последствия | Согласие добавлено в систему.                                               |

## Сценарий

### Основной поток

(1) Браузер запрашивает станицу "Новое согласие".

(2) Оператор перешел на страницу "Новое согласие".

(3) Оператор вводит данные согласия и нажимает кнопку "Сохранить".

(4) Система добавляет согласие в базу данных.

(5) Система перенаправляет администратора на страницу со списком согласий.
